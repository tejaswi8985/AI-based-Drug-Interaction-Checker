# 💊 AI-Based Drug Interaction Checker

## 📌 Overview

The AI-Based Drug Interaction Checker is a web application designed to analyze and detect potential interactions between different drugs. It leverages AI techniques to provide insights that can help users make safer healthcare decisions.

---

## 🚀 Features

* 🔍 Check interactions between multiple drugs
* 🤖 AI-powered analysis for accurate results
* 🌐 User-friendly web interface
* ⚡ Fast and scalable performance

---

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript, Vite
* **Backend:** Python
* **AI Integration:** OpenAI API
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/AI-based-Drug-Interaction-Checker.git
cd AI-based-Drug-Interaction-Checker
```

### 2️⃣ Setup backend

```bash
cd backend
pip install -r requirements.txt
```

### 3️⃣ Setup frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend and add:

```
OPENAI_API_KEY=your_api_key_here
```

⚠️ Do NOT upload `.env` to GitHub.
